# تعليمات إضافة ملف الصوت للإشعارات

## الملف المطلوب:
- المسار: `dashboard_app/assets/sounds/new_order_notification.mp3`
- النوع: ملف صوتي MP3
- المدة: 1-2 ثانية
- النوع: صوت إشعار واضح ومميز

## بدائل إذا لم يتوفر الملف:
1. استخدام صوت النظام الافتراضي (SystemSoundType.alert)
2. استخدام أي ملف صوتي قصير ومميز

## ملاحظة:
الكود جاهز للعمل مع أو بدون الملف الصوتي. إذا لم يتوفر الملف، سيتم استخدام صوت النظام كبديل تلقائياً.

## اختبار الصوت:
- اضغط على زر "اختبار الصوت" في شاشة إدارة الطلبات
- أو انتظر وصول طلب جديد لسماع الصوت تلقائياً
