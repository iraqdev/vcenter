# تعليمات تشغيل تطبيق الداشبورد

## المتطلبات الأساسية
- Flutter SDK (الإصدار 3.9.2 أو أحدث)
- Android Studio أو VS Code
- حساب Firebase مع مشروع v-center-5f74b

## خطوات التشغيل

### 1. إعداد Firebase
```bash
# انتقل إلى مجلد الداشبورد
cd dashboard_app

# تثبيت المكتبات
flutter pub get
```

### 2. إعداد قاعدة البيانات
تأكد من أن قاعدة البيانات Firebase تحتوي على:
- مجموعة `users` مع الحقول التالية:
  - `name` (String)
  - `phone` (String) 
  - `city` (String)
  - `address` (String)
  - `near` (String)
  - `point` (Number)
  - `active` (Boolean)
  - `isReviewed` (Boolean)
  - `createdAt` (Timestamp)
  - `updatedAt` (Timestamp)
  - `shopLocation` (Map - اختياري)
  - `profilePic` (String - اختياري)
  - `shopPic` (String - اختياري)

### 3. تشغيل التطبيق
```bash
# تشغيل على Android
flutter run

# تشغيل على الويب
flutter run -d chrome

# تشغيل على Windows
flutter run -d windows
```
