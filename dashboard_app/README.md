# VCenter Dashboard

لوحة تحكم متقدمة لإدارة تطبيق VCenter

## المميزات

### إدارة المستخدمين
- ✅ عرض جميع المستخدمين المسجلين
- ✅ البحث في المستخدمين
- ✅ تعديل معلومات المستخدمين
- ✅ تفعيل/حظر المستخدمين
- ✅ حذف المستخدمين
- ✅ إحصائيات المستخدمين

### مراجعة الحسابات الجديدة
- ✅ عرض الحسابات الجديدة غير المراجعة
- ✅ مراجعة تفاصيل كل حساب جديد
- ✅ الموافقة على الحسابات الجديدة
- ✅ رفض الحسابات الجديدة
- ✅ إزالة الحسابات من قائمة المراجعة

## التقنيات المستخدمة

- **Flutter** - إطار العمل الرئيسي
- **GetX** - إدارة الحالة والتنقل
- **Firebase Firestore** - قاعدة البيانات
- **Firebase Storage** - تخزين الصور
- **Font Awesome** - الأيقونات

## التثبيت والتشغيل

1. تأكد من تثبيت Flutter SDK
2. انتقل إلى مجلد المشروع:
   ```bash
   cd dashboard_app
   ```
3. تثبيت المكتبات:
   ```bash
   flutter pub get
   ```
4. تشغيل التطبيق:
   ```bash
   flutter run
   ```

## الاستخدام

### الشاشة الرئيسية
- عرض إحصائيات المستخدمين
- الوصول السريع لإدارة المستخدمين
- الوصول لمراجعة الحسابات الجديدة

### إدارة المستخدمين
- عرض قائمة بجميع المستخدمين
- البحث في المستخدمين بالاسم أو الهاتف
- تعديل معلومات أي مستخدم
- تفعيل أو حظر المستخدمين
- حذف المستخدمين

### مراجعة الحسابات الجديدة
- عرض الحسابات الجديدة التي تحتاج مراجعة
- مراجعة تفاصيل كل حساب جديد
- الموافقة على الحسابات المقبولة
- رفض الحسابات غير المقبولة
